== Installation ==

To use Invoicepress, you will need:

* 	an installed and configured copy of [WordPress][]	(version 3.0 or later).

*	FTP, SFTP or shell access to your web host

= New Installations =

Method-A:

1. Go to your wordpress admin "yoursite.com/wp-admin"

2. Login and then access "yoursite.com/wp-admin/plugin-install.php?tab=upload

3. Upload and activate this plugin

4- That's it, now see your dashboard and unser settings you will see Invoicepress sub-menu item. Alternatively you will see this menu item under WooCommerce menu if WooCommerce was installed and active.


Method-B:

1.	Download the Invoicepress installation package and extract the files on	your computer. 

2.	Create a new directory named `invoicepress` in the `wp-content/plugins`	directory of your WordPress installation. Use an FTP or SFTP client to upload the contents of your Invoicepress archive to the new directory that you just created on your web host.

3.	Log in to the WordPress Dashboard and activate the Invoicepress plugin.

4.	Once the plugin is activated, Invoicepress will be displayed under settings menu or alternatively under WooCommerce menu if WooCommerce is installed and activated.

= Upgrades =

To *upgrade* an existing installation of Invoicepress to the most recent release:

1.	Download the Invoicepress installation package and extract the files on your computer. 

2.	Upload the new PHP files to `wp-content/plugins/invoicepress`,	overwriting any existing Invoicepress files that are there.

3.	Log in to your WordPress administrative interface immediately in order to see whether there are any further tasks that you need to perform to complete the upgrade.

4.	Enjoy your newer and hotter installation of Invoicepress
